# 🧠 What is CharacterForge AI?

**CharacterForge AI** is a SaaS-style AI platform that generates fully immersive fictional characters with:

- 🎭 Names & identities  
- 🧬 Detailed personalities  
- 📖 Rich backstories  
- ⚔️ Powers & abilities  
- 💬 Interactive AI chat  
- 🎨 AI-generated portraits (Pollinations AI / fallback system)

---

# 💡 Key Features

## 🎭 AI Character Generator
- Fantasy / Sci-Fi / Horror / Cyberpunk support  
- Deep storytelling generation  
- Unique personalities every time  

## 💬 AI Roleplay Chat
- Talk to your characters in real-time  
- Personality-aware responses  
- Immersive roleplay mode  

## 🎨 Image Generation System
- Pollinations AI (primary)  
- Automatic fallback avatar system  
- Always displays a character image (no blank UI)

## ⚡ Instant SaaS Experience
- No setup required  
- Works directly in browser  
- One-click generation

# 🧠 How It Works

### 1️⃣ User Input
- Genre
- Personality
- Powers

### 2️⃣ AI Engine (Groq)
Generates:
- Character profile
- Backstory
- Abilities
- Personality traits

### 3️⃣ Image Engine
- Pollinations AI generates portrait  
- If failure → DiceBear fallback avatar  

### 4️⃣ Chat Engine
- AI roleplay system using Groq LLM  
- Character stays in persona
